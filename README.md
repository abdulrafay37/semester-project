# Blood Donor & Emergency Finder System
## Overview
The Blood Donor & Emergency Finder System is a Java console-based application designed to help manage blood donor information and quickly find suitable donors during emergencies.
This project was developed using basic Java programming concepts such as arrays, methods, loops, recursion, switch statements, conditional statements, and user input handling.
## Features
* Add new blood donors
* View all registered donors
* Search donors by blood group
* Emergency blood request search
* Display total donor count
* Recursive donor searching
* Menu-driven interface
## Technologies Used
* Java
* Scanner Class
* Arrays
* Methods
* Loops
* Recursion
* Switch Statements
## Project Structure
BloodDonorSystem.java
## How to Run
1. Install Java JDK.
2. Compile the program:
```bash
javac BloodDonorSystem.java
```
3. Run the program:
```bash
java BloodDonorSystem
```
